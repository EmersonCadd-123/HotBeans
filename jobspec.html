Job Specification

Website Creator:

In this role, you are the creator of the website, you will recieve the design and need to be able to code the website to the design standard, to be able to satisfy clients.

Website Tester:

In this role, you will need to be able to recieve the website and be able to test the speed of the website, any errors and features that needs to be added to the website, to ensure that the website can be send of to the client with no issues.

Requirements: experience in coding

Requirements: be able to identify certain issues, communication skills, experience in designing

Website Designer:

In this role, you will be with the client and he will ask what he wants in the website, what you need to do is be able to create potential design for the client to be able to give the client options on how the website should be .

Requirements: Creativity, communication, experience in desgning.
